# Laravel Service
Exemplo de criação de um serviço com Docker e utilizando Laravel.

Configuração inicial
-----------

#### Link do [Docker](https://docs.docker.com/).
```console
~$ docker-compose up -d
~$ docker-compose ps
~$ docker-compose exec productapp bash
~$ composer update
~$ php artisan migrate
```

Host de acesso: `http://localhost:8001`
